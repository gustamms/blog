# Bem-vindo ao meu blog!

Este é um exemplo de como estruturar posts no formato markdown.

## Como funciona

Cada arquivo `.md` na pasta `posts/` se torna um post no site. O frontmatter contém:

- **title**: título do post
- **description**: breve descrição
- **tags**: tags para facilitar a busca
- **date**: data de publicação

## Markdown

Você pode usar **markdown** normalmente nos posts:

```typescript
const nome = "Gustavo";
console.log(`Olá, ${nome}!`);
```
